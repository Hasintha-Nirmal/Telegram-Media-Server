## Recent Updates

### Privacy & Security
- **Per-Account Database Isolation** - Each account has its own database for complete privacy
- **Multi-Account Support** - Create, upload, and switch between multiple Telegram accounts
- **Session Management** - Web-based session creation with 2FA support

### Features
- **3-Step Session Creation** - Phone → Code → 2FA (if needed)
- **Background Workers** - Auto-sync and download queue
- **Video Streaming** - HTTP range requests for progressive playback
- **Smart Downloads** - Queue-based parallel downloads
